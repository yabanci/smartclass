# Smart Classroom — инструкция по установке и использованию

Эта инструкция написана для тех, кто **не работает в IT**. Всё, что нужно — уметь скачивать программы и копировать команды. Займёт **около 30 минут**.

---

## Что такое Smart Classroom

Это приложение для управления умными устройствами в классе: включать/выключать свет, кондиционер, розетки, проектор со смартфона или с компьютера учителя. Всё делается одной кнопкой или по расписанию уроков.

---

## Часть 1. Подготовка компьютера

Вам понадобится:

- **Компьютер с Windows 10/11 или macOS** (минимум 8 ГБ оперативной памяти)
- **Интернет** для скачивания программ (потом можно работать без него)
- **Домашняя или школьная Wi-Fi** (для подключения умных устройств)

### Шаг 1. Установить Docker Desktop

Docker — это программа, которая запустит наше приложение. Ставится один раз.

1. Зайдите на сайт: **https://www.docker.com/products/docker-desktop/**
2. Нажмите **"Download Docker Desktop"**
3. Выберите свою систему:
   - Если у вас **Windows** → "Download for Windows (AMD64)"
   - Если у вас **Mac с процессором Apple M1/M2/M3** → "Download for Mac - Apple Silicon"
   - Если у вас **Mac с процессором Intel** → "Download for Mac - Intel Chip"
4. Запустите скачанный файл и следуйте инструкциям установщика
5. После установки **перезагрузите компьютер**
6. Найдите в меню "Пуск" (Windows) или в "Launchpad" (Mac) иконку **Docker Desktop** и запустите её
7. Подождите 1-2 минуты, пока внизу справа не появится надпись **"Docker is running"** или зелёный значок

**Важно:** Docker Desktop должен быть запущен всё время, пока вы пользуетесь нашим приложением. Если компьютер перезагружается — Docker надо запустить снова (или настроить автозапуск в его настройках).

### Шаг 2. Установить Git

Git нужен, чтобы скачать проект с интернета.

**Windows:**
1. Скачайте: **https://git-scm.com/download/win**
2. Запустите установщик
3. При установке нажимайте **"Next"** на каждом шаге, ничего менять не надо
4. В конце нажмите **"Finish"**

**Mac:**
1. Откройте приложение **"Терминал"** (его можно найти через Spotlight: `Cmd+Space` → наберите "Terminal")
2. Вставьте команду и нажмите Enter:
   ```
   git --version
   ```
3. Если увидите запрос установить — нажмите **"Install"**
4. Если увидите номер версии (например `git version 2.39.0`) — Git уже стоит

## Часть 2. Скачать приложение

### Шаг 3. Открыть командную строку

**Windows:**
- Нажмите кнопку **Пуск** → наберите **"PowerShell"** → откройте **"Windows PowerShell"**

**Mac:**
- Откройте **"Терминал"** (как в шаге 2)

### Шаг 4. Скачать проект

Вставьте в PowerShell или Терминал следующие команды **по одной** (скопируйте одну, вставьте, нажмите Enter, дождитесь завершения, потом следующую):

Перейти в папку "Документы":
```
cd Documents
```

Скачать проект:
```
git clone https://github.com/yabanci/smartclass.git
```

Перейти в папку проекта:
```
cd smartclass
```

Всё, проект скачан.

## Часть 3. Запустить приложение

### Шаг 5. Первый запуск

Убедитесь, что **Docker Desktop запущен** (внизу справа зелёный значок).

В PowerShell или Терминале введите:
```
docker compose up -d --build
```

Нажмите Enter. Подождите **5-10 минут** — компьютер скачивает компоненты и запускает их. На экране будет много текста, это нормально.

Когда всё готово, вы увидите строки вида:
```
Container smartclass-postgres  Started
Container smartclass-backend   Started
Container smartclass-frontend  Started
Container smartclass-hass      Started
```

### Шаг 6. Создать тестовых пользователей

Ещё одна команда — чтобы в системе появился учитель и класс для пробы:
```
python scripts/seed.py
```

Если выдаст ошибку "python не найден" — скачайте Python: **https://www.python.org/downloads/** (при установке обязательно поставьте галочку **"Add Python to PATH"**). Потом снова запустите команду.

### Шаг 7. Подождать, пока Home Assistant полностью запустится

Home Assistant — это "мотор" для умных устройств. При первом запуске ему нужно **2-5 минут**, чтобы подготовиться. Пока просто подождите.

Проверить готовность можно командой:
```
docker compose logs backend --tail 5
```

Если видите строку `hass: onboarded and token persisted` — всё готово. Если нет — подождите ещё минуту и повторите команду.

## Часть 4. Первый вход

### Шаг 8. Открыть приложение

Откройте браузер (Chrome, Edge, Safari) и зайдите по адресу:
```
http://localhost:3000
```

Вы увидите экран входа.

### Шаг 9. Войти как учитель

- **Email:** `teacher@smartclass.kz`
- **Пароль:** `teacher1234`

Нажмите **"Войти"**.

После входа попадёте в класс "Кабинет 101" с тремя демо-устройствами. Это для того, чтобы сразу увидеть, как всё работает.

**На главной странице теперь есть:**
- **"Быстрое управление"** — кнопки «Все включить» / «Все выключить» / «Эко»: одним нажатием управляют всеми устройствами кабинета сразу.
- **Статистика сверху** — сколько устройств онлайн, оценка нагрузки в кВт, температура и влажность.
- **Текущий урок** — подсвечен зелёным, если идёт сейчас.

**В шапке приложения:**
- Иконка **луны/солнца** — переключает светлую/тёмную тему. Настройка запоминается.
- Иконка **колокольчика** — уведомления (перегрев в классе, устройство ушло в офлайн и т.п.). Красный кружок показывает, сколько непрочитанных.

## Часть 5. Подключить реальное умное устройство

Рассмотрим на примере умной розетки **Xiaomi Mi Smart Plug** — самый распространённый случай.

### Шаг 10. Подключить розетку в приложении Mi Home

Это делается **на телефоне**, один раз.

1. Скачайте на телефон приложение **Mi Home** (App Store или Google Play)
2. Зарегистрируйтесь: введите email или номер телефона, придумайте пароль. **Запомните, какой регион выбрали** (Европа, Россия, Сингапур, Казахстан — неважно какой, главное помните)
3. **Телефон должен быть подключён к 2.4 ГГц Wi-Fi** (не 5 ГГц). В Xiaomi-роутере AX3000 отключите "Smart Connect" перед этим шагом
4. Воткните розетку в электросеть. На ней замигает жёлтая лампочка — она готова к настройке
5. В Mi Home нажмите **"+"** → **"Add device"** → телефон сам найдёт розетку
6. Введите пароль от домашней Wi-Fi → дождитесь "Successfully added"
7. В приложении Mi Home нажмите на розетку — должно раздаться "щёлк", лампочка загорается синим. Значит работает

### Шаг 11. Подключить розетку в Smart Classroom

1. В браузере на http://localhost:3000 → войдите как учитель
2. Снизу нажмите вкладку **"Устройства"**
3. Вверху справа нажмите кнопку с иконкой радара — **"Найти IoT"**
4. Откроется окно с каталогом брендов. Нажмите карточку **"Xiaomi · Mi Home · Yeelight"**
5. Прочитайте подсказку → **"Далее"**
6. Откроется форма. Это и есть подключение через Xiaomi:
   - Поставьте галочку в **"eula"** (согласие с условиями)
   - Нажмите **"Далее"**
7. Откроется новая вкладка браузера с формой Xiaomi. Войдите под тем **email и паролем**, которыми регистрировались в Mi Home
8. **ВАЖНО** (только для Windows): после успешного входа Xiaomi попробует вернуть вас на адрес `homeassistant.local:8123`. Windows этот адрес не понимает, браузер покажет ошибку "Не удаётся открыть страницу"
9. В адресной строке замените `homeassistant.local` на `localhost` и нажмите Enter. Пример:
   - Было: `http://homeassistant.local:8123/api/webhook/...`
   - Стало: `http://localhost:8123/api/webhook/...`
10. После этого откроется форма "Импорт семьи устройства" — поставьте галочку напротив вашего дома (обычно "Home"), выберите "название комнаты (Спальня)" и нажмите "Submit"
11. Вернитесь в наше приложение на http://localhost:3000 → вкладка "Устройства" → снова "Найти IoT"
12. В разделе **"Найденные устройства"** внизу будет ваша розетка — нажмите **"Добавить в кабинет"** рядом с ней
13. Появится зелёная надпись **"Устройство отвечает"** — готово!

### Шаг 12. Пользоваться

Закройте окно "Найти IoT". В списке устройств класса теперь есть ваша розетка. Рядом с ней — **переключатель-тумблер** (серый/зелёный). Нажмите — розетка щёлкнет и тумблер загорится зелёным.

**Управление значениями прямо из карточки устройства:**
- **Кондиционер / климат-контроль** — ползунок температуры **16-30 °C**. Двигайте до нужной температуры — устройство получит команду.
- **Свет / диммер** — ползунок яркости **0-100 %**.
- **Жалюзи / шторы** — ползунок положения **0-100 %**.
- **Вентилятор / освежитель воздуха** — три кнопки **Слабо / Средне / Сильно**.

Дополнительно у каждой карточки:
- Иконка **карандаша** — редактировать параметры устройства (название, драйвер, конфиг).
- Иконка **корзины** — удалить устройство из кабинета.

Сверху страницы **"Устройства"** тоже есть блок «Быстрое управление» с кнопками «Все включить» / «Все выключить» — удобно перед уроком и после.

## Часть 6. Повседневное использование

### Каждый день

Ничего специально делать не нужно. Docker Desktop запускается вместе с компьютером (если включили автозапуск в его настройках). Приложение работает автоматически.

Если хотите **остановить** приложение (например, на ночь) — в командной строке:
```
docker compose down
```

Чтобы **снова запустить**:
```
docker compose up -d
```

### Посмотреть, работает ли приложение

```
docker compose ps
```

Должно быть 5 строк со статусом **"Up"** или **"healthy"**.

### Смена пароля учителя

В приложении: нижняя вкладка **"Профиль"** → **"Сменить пароль"**.

### Добавить ещё один класс

1. Вкладка **"Главная"** → сверху синяя кнопка **"Новый кабинет"**
2. Введите название (например "Кабинет 205") → **"Создать"**
3. Сверху в шапке можно переключаться между классами

### Расписание уроков

1. Нижняя вкладка **"Расписание"**
2. Нажмите **"Добавить урок"** → предмет, день, время начала/конца
3. Создайте **"Сценарий"** (например "Начало урока: включить свет, открыть шторы") во вкладке Сценарии
4. Укажите, что сценарий запускается в начале урока — свет сам включится

## Часть 7. Что делать, если не работает

### "docker compose" не распознаётся как команда

Проверьте, что Docker Desktop **запущен** (значок Docker в панели задач/меню). Если значок серый или отсутствует — запустите Docker Desktop и подождите, пока он полностью загрузится (2-3 минуты).

### Страница http://localhost:3000 не открывается

1. Проверьте: `docker compose ps` — все 5 контейнеров должны быть "Up"
2. Если нет — перезапустите: `docker compose restart`
3. Если один из контейнеров постоянно падает — напишите разработчику с выводом команды `docker compose logs`

### "Home Assistant не отвечает"

Это нормально в первые 2-5 минут после запуска. Просто подождите. Если ошибка держится дольше 10 минут — выполните:
```
docker compose restart homeassistant
```

### Mi Home не находит розетку

- Телефон должен быть подключён к **2.4 ГГц Wi-Fi**, не к 5 ГГц
- На некоторых роутерах объединены сети 2.4 и 5 ГГц — отключите эту функцию временно
- Удерживайте кнопку на розетке 5 секунд, чтобы сбросить её к заводским (лампочка начнёт быстро мигать жёлтым)

### При подключении Xiaomi в наше приложение вылетает "cloud_login_error"

Это значит, что Xiaomi сломал старую версию их API. Мы уже подключили новую — интеграция **"Xiaomi · Mi Home · Yeelight"** в нашем визарде использует её автоматически. Если всё равно ошибка — перезапустите всё:
```
docker compose down
docker compose up -d --build
```
Подождите 5 минут и попробуйте снова.

### Забыли пароль `teacher1234`

Если успели поменять и не помните — сброс возможен только через полный ресет базы (удалятся все данные):
```
docker compose down -v
docker compose up -d
```
Подождите 5 минут, потом `python scripts/seed.py` — пользователи и пароли восстановятся по умолчанию.

### Занят порт 5432 / 8080 / 3000

Обычно это значит, что у вас уже запущено другое приложение на этом порту (чаще всего PostgreSQL или какой-то локальный сайт). Выключите его или обратитесь к разработчику — он подскажет, как сменить порт.

## Часть 8. С телефона

Приложение открывается и на смартфоне, если телефон в **той же Wi-Fi сети**, что и компьютер.

1. Узнайте IP-адрес вашего компьютера в локальной сети:
   - **Windows**: PowerShell → `ipconfig` → ищите "IPv4 Address" (обычно вида `192.168.1.50`)
   - **Mac**: Настройки → Wi-Fi → рядом с вашей сетью "Подробности" → IP-адрес
2. На телефоне откройте Safari / Chrome и введите: `http://192.168.1.50:3000` (подставьте свой IP)
3. Войдите как учитель → пользуйтесь

**Чтобы работало как приложение с иконкой:**
- **iPhone/iPad:** в Safari нажмите значок "Поделиться" → **"На экран Домой"**
- **Android:** в Chrome нажмите меню (три точки) → **"Добавить на главный экран"**

## Приложение. Список команд-шпаргалок

Все команды вводятся в PowerShell (Windows) или Терминале (Mac), **находясь в папке проекта** (`cd Documents/smartclass`).

| Что хотим | Команда |
|---|---|
| Запустить | `docker compose up -d` |
| Остановить | `docker compose down` |
| Перезапустить | `docker compose restart` |
| Посмотреть статус | `docker compose ps` |
| Посмотреть логи | `docker compose logs --tail 50` |
| Обновить код | `git pull` потом `docker compose up -d --build` |
| Полный сброс (удаляет ВСЕ данные) | `docker compose down -v` |
